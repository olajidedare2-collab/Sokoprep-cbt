Sokoprep CBT Platform

Sokoprep is a Computer-Based Test (CBT) platform designed to help civil servants and students prepare effectively for exams.

🌐 Live Website: https://sokoprep.com

🚀 Features
Online CBT examination system
Timer-based exams
Question upload and management
User authentication (login system)
Dashboard for exam tracking
🛠️ Technologies Used
PHP
MySQL
HTML/CSS
JavaScript
📂 Project Structure
index.php – Homepage
login.php – User login
dashboard.php – User dashboard
exam.php – Exam interface
add_question.php – Add questions
import_questions.php – Bulk upload
db.php – Database connection
🎯 Purpose

Sokoprep is built to:

Help users practice CBT exams
Prepare for civil service promotions
Provide an accessible online testing platform
🔮 Future Improvements
Advanced result analytics
Admin control panel upgrade
Mobile optimization
Payment integration (Paystack)
